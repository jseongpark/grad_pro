CNN

: 일차적으로, 특수문자를 통한 욕설 필터링 우회시도에 대해 이미지 처리를 통해 한글로 복원한다.

RoBERTa

: 이후 금칙어 사전을 통한 필터링으로 거르지 못하는 욕설을 문맥으로 판단한다.

  • VGG16 구조를 기반으로 독자적인 모델을 생성하여 사용.
  • 입력된 텍스트 중 한글과 띄어쓰기를 제외한 다른 텍스트를 이미지 처리 후 CNN으로 인식. CNN으로 인식된 텍스트를 오토마타를 이용하여 한글 음절로 재조립.
  • 텍스트를 단순한 데이터 구조가 아닌 인간이 인지하는 형태로 변환하여 RoBERTa에서 판단을 할 때, 보다 인간의 판단에 가까운 판단을 할 수 있도록 변경하는 역할을 수행.
  • 기존 RoBERTa 모델은 보통 영어를 기반으로 학습 되었기에 한글 기반의 토크나이저를 새로 학습 시켜야 했음
  • Mecab 형태소 분석기를 이용해 약 3만개의 데이터 셋의 단어들을 분리하고, 그것을 BertWordPieceTokenizer에 학습시킴
  • 그리고 해당된 RoBERTa에 넣어 문맥을 고려한 욕설을 판단할 수 있도록 하였다
